- Can store duplicates (==non-unique==)

- Stores and outputs in ==increasing order== by default (can be changed)

- Fast insertion, deletion & search operations

 
![[Excalidraw/STL.md#^4fnQWKiZ]] ^swap

Used to exchange the contents of two multisets but multisets must be of the same type, although sizes may differ.

![[Excalidraw/STL.md#^NqE3oyU0]] ^lowerbound

Find the first element in the multiset that is equal to or greater than the given value.

![[Excalidraw/STL.md#^5nLcJcA2]] ^upperbound

Find the first element in the multiset that is just greater than the given value.

![[Excalidraw/STL.md#^3ye93Vkc]] ^getallocator

Returns a copy of the allocator object associated with the multiset.

![[Excalidraw/STL.md#^1bZjjijk]] ^equalrange

- Returns an iterator of pairs.  
- The pair refers to the range that includes all the elements in the container which have a key equivalent to k.
