- ==Unique elements only==.

- sorted automatically in ascending order.

- Can be added or removed, but the value of an existing element cannot be changed (==unmutable==).

- ==Cannot be accessed by index numbers==, because the order is based on sorting and not indexing.

`merge()` ^merge

Merge one set into another set.

`contains()` ^contains

used to find out whether a particular element is present or not.

`insert_range()` ^insertrange

Insert multiple elements at the same time in set.

`key_comp()` ^keycomp

Returns a copy of the comparison object used by the container.
