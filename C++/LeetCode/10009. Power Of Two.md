
Given an integer `n`, return _`true` if it is a power of two. Otherwise, return `false`_.

An integer `n` is a power of two, if there exists an integer `x` such that `n == 2x`.

**Constraints:**

- `-231 <= n <= 231 - 1`

**Follow up:** Could you solve it without loops/recursion?


Solution:

1) Divide n by 2 till quotient = 1: Every even integer will reach 1 at one point ---> Wrong

2) initialize a variable "ans" to 1 
	- run loop for i from 0 to 30
	 - at the end of the loop, multiply ans by 2    // check if it is exceeding INT_MAX before the multiplication
	 - before the multiplication everytime, check if ans has reached a value equal to n? if yes --> true, if no ---> multiply by 2, repeat until its going to exceed INT_MAX
	 - if ans reaches final value and still isnt equal to n ---> false
	 
```cpp

bool isPowerOfTwo(int n) {

    if (n <= 0) return false;

    int ans = 1;

    for (int i = 0; i <= 30; i++)

    {

  

        if (ans == n)

        {

            return true;

        }

  

    if (ans > INT_MAX/2){break;}

        ans = ans*2;

    }

    return false;

    }

```
